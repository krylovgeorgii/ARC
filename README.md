# ARC

test_merged.cpp

mobile robotics <br>
for myRIO-1900(1950) controller from National Instruments <br>
based on ARM Cortex-A9 and FPGA Zynq-7010 <br>
with linux realtime onboard 

----

test_vision.cpp

computer vision <br>
for linux <br>
need installed SDL2 and jpeglib <br>
use usb-camera YUV 4:2:2 format
