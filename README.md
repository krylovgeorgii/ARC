# ARC

main.cpp

mobile robotics 
for myRIO-1900(1950) controller from National Instruments 
based on ARM Cortex-A9 and FPGA Zynq-7010
with linux realtime onboard

----

test_vision.cpp

computer vision 
for linux
need installed SDL2 and jpeglib
use usb-camera YUV 4:2:2 format
